# node-minimal

A minimal [Node.js](http://nodejs.org/) app using [Express 4](http://expressjs.com/).

## Running the App

```sh
$ git clone https://github.com/johncraw/node-minimal.git
$ cd node-minimal
$ npm install
$ npm start
```

The minimal app should now be running on [localhost:3000](http://localhost:3000/).
